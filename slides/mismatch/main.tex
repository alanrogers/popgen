\begin{frame}
\frametitle{What is a mismatch distribution?}

Count the number of site differences between each pair of sequences in 
a sample, and use the resulting counts to build a histogram.  You end
up with a ``mismatch distribution.''  The $i$th entry of the mismatch
distribution is the number of pairs of sequences that differ by $i$
sites.
\end{frame}

\begin{frame}
At mutation-drift equilibrium, a random pair of sequences differs by
$i$ sites with probability 
\begin{equation}
F_i = \left(\frac{1}{\theta + 1}\right)
\left(\frac{\theta}{\theta + 1}\right)^i, \qquad (i=0,1,2,\ldots)
\end{equation}
\mbox{}\hfill (Watterson, 1975)\\
\end{frame}

\begin{frame}
\frametitle{Why does the stationary neutral model fit human data so poorly?}
There are several hypotheses to consider:
\begin{enumerate}
\item Sampling error. (Important because the pairs of genes in our sample are
  correlated.) 
\item Selection.
\item Failure of infinite sites hypothesis.
\item Non-random mating.
\item Variation in population size. 
\end{enumerate}
Work has been done on all of these possibilities, but I will only try
to tell you about the last one.
\end{frame}

\begin{frame}
\frametitle{Coalescent theory in a population of varying size}

At any given time, $t$, the hazard of a coalescent event is 
\[
h_i(t) = \frac{i(i-1)}{4N(t)}
\]
But $N(t)$ is no longer constant.

\bigskip
\[
E[t_i] \neq 1/h_i
\]
We need computer simulations.
\end{frame}

\begin{frame}
  \frametitle{Model of sudden change in population size}

  Time variable: $\tau = 2\mu t$, where $\mu$ is the mutation rate and
  $t$ is time in generations.

  \medskip

  Population size changes suddenly at time 0. $\theta = 4N\mu$
  measures population size after this change.

  \medskip

  At time 0, $F_i(0)$ is probability that a random pair of sequences
  differ by $i$ sites. At time $\tau$, this probability becomes
  $F_i(\tau)$. Converges toward an equilibrium at which $F_i(\tau) =
  \hat F_i$.

  \medskip

  At time $\tau$,
  \[
F_i(\tau) = \hat F_i + e^{-\tau(1+1/\theta)}
  \sum_{j=0}^i\frac{\tau^j}{j!}\big(F_{i-j}(0)  - \hat F_{i-j}\big).
  \]
\mbox{}\hfill (Li, 1977; Rogers \& Harpending, 1992)\\  
\end{frame}
